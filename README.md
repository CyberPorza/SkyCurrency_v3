# 🚀 SkyCurrency Pro v3.0

Modern, fast, and light-weight Financial Terminal built with Python Flask.

## 📊 Features
- **Compact UI:** Weather-bar and split-screen terminal design.
- **Global Markets:** Live rates for USD, EUR, GBP, CHF, and more.
- **Detailed Gold:** Live scraping for Gram, Quarter, Half, and Full Gold prices.
- **Real-time:** Integrated JavaScript clock and auto-refresh.

## 🛠️ Tech Stack
- **Backend:** Flask, Requests, BeautifulSoup4
- **Frontend:** Bootstrap 5, FontAwesome
- **Deployment:** Optimized for Render.com

## ⚙️ Deployment
1. Connect GitHub to **Render.com**
2. **Build Command:** `pip install -r requirements.txt`
3. **Start Command:** `gunicorn app:app`
